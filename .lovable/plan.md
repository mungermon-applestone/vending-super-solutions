

# Add NAMA Trade Show Banner

Add a new entry to the `TRADE_SHOW_CONFIGS` in `src/hooks/useTradeShowVisitor.ts`:

```
nama: {
  param: 'nama',
  message: "Welcome, NAMA visitors, thanks for stopping by our booth!",
}
```
